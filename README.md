# FastAPI Project
## How to run?

1. Download
2. Activate venv, on w10 with this command in cmd: path_to_root\fastapi_env\Scriptsactivate.bat
3. Enter the following command in cmd o powershell: pip install -r requirements.txt
4. To launch app run: uvicorn blog.main:app --reload
5. Enter in your browser: http://127.0.0.1:8000/docs
6. Create a user with the post method in the User section
7. Authentificate user to generate token
8. Enjoy
